# CODSOFT
# Mobile Signup Flow

## Overview

This project is a responsive Mobile Signup Flow designed and developed using HTML, CSS, and JavaScript. It provides a modern user interface for user registration and login with form validation and interactive navigation between screens.

## Features

* Welcome Screen with onboarding image
* User Signup Form
* User Login Form
* Email Validation
* Password Validation
* Show/Hide Password Functionality
* Privacy Policy Agreement Checkbox
* Responsive Mobile UI Design
* Smooth Screen Transitions
* Modern and Clean User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```
├── index.html
├── styles.css
├── app.js

└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.
4. Explore the Welcome, Signup, and Login screens.
